01 — About

I'm a Java engineer based in Amersfoort, Netherlands, with over 14 years delivering enterprise-grade backend applications. My clients have included KPN, ABN Amro Bank, FedEx Express, TomTom, ASML, Siemens, Bharti AXA, and Delta Airlines — organisations where reliability and uptime are non-negotiable.

My core is Java 17 + Spring Boot 3 + microservices, wired into cloud-native infrastructure on Azure AKS. I've led architecture teams at FedEx, designed Kafka-driven event pipelines, and built CI/CD practices across Angular frontends and Docker-based deployments.

I hold a BE in Computer Science from JSS Academy (Visveswaraya Technological University, 2011). After a decade building foundations in India across Cognizant, Siemens, and Capgemini, I moved to the Netherlands and have been delivering complex backend applications for Dutch enterprise ever since.

// How I approach system design

THINKING
01
Contract before code

OpenAPI/Swagger specs and Kafka topic schemas are the first deliverable. Every team member knows the boundary before implementation starts.

02
Design for failure first

Circuit breakers, dead-letter queues, retry policies. I've seen what happens without them at FedEx logistics scale — you don't get a second chance.

03
Containerise early, automate everything

Docker from day one. Kubernetes manifests alongside the service. Jenkins/Azure DevOps pipelines before the first PR is merged.

04
Quality gates enforced, not requested

JUnit 5 + Mockito coverage and SonarQube gates in every pipeline — not left to the honour system or sprint end reviews.
